# Contact-attempt templates and checklist

The Plugin Review team requires "all reasonable efforts" to reach the
original developer before an adoption request. Make these attempts, record
the date and URL of each, and screenshot them. Wait at least 2–4 weeks for
replies before sending adoption-email.md.

## Checklist

- [ ] Support-forum post (template below) — record date + link
- [ ] GitHub issue on the original upstream repo (template below) — record date + link
- [ ] Email, if an address can be found (old plugin readmes, WHOIS history
      for nomiddlemancrypto.io, the `rgostic` wordpress.org profile page)
- [ ] Screenshot each attempt
- [ ] Create/verify your wordpress.org account (needed to receive commit access)

## Notes

- Keep the tone factual; the review team reads these threads.
- The 30-day clock the team gives the author starts only after your code
  passes review — your earlier attempts just establish good faith.
- If the author replies and agrees: they can add you as a committer
  themselves under Advanced View → Committers on the plugin page, which
  skips the whole formal process.
